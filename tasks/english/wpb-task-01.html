Task01 - Hello F#

Let's start with the basics:

  • In a .fsx script:

    Highlight the '10' and press Alt+Enter, which will open a console and give the output 'val it: int = 10'. (If the code to execute has only one line, you can just click into the line and press Alt+Enter without highlighting.)

  • In a notebook:

    There is a 'play' button in the left upper corner of the field below. Press it to execute the respective code block, which in this case will simply give '10' as output.

    Note that you always have to end codeblocks in notebooks with the value you want to put out

10

Working in this interactive way has the advantage that you can always directly test your code and look at the output. Try it with the line below and appreciate that the result of the calculation is 30.

10 + 20

Everything after a // or inside of (* ... *) is treated as a comment and will not get executed as code. You can use (* ... *) to comment parts of a single line. The standard keybinding for commenting a selection is (Strg + K -> Strg + C)

10 (*+ 40 *) + 30

If you not quite get a task right, please submit partial solutions or trails of thought as comments to get partial points!
